# MoreFood

一个基于 **Fabric** 的 Minecraft 食物扩展模组。  
给原版加了一批新食物、苹果分支、肉类加工食物，以及一些额外内容，比如自定义效果、方块、武器、生物和结构。

## 兼容信息

| 项目 | 版本 |
| --- | --- |
| Minecraft | `1.21.7` |
| Fabric Loader | `0.18.4+` |
| Fabric API | `0.129.0+1.21.7` |
| Java | `21` |
| Mod ID | `more_food` |

## 目前内容

### 1. 苹果系列

模组目前加入了多种特殊苹果：

- 青苹果
- 钻石苹果
- 绿宝石苹果
- 红石苹果
- 黑曜石苹果
- 末影苹果
- 下界合金苹果
- 雷电苹果
- 无限苹果
- 月读苹果

其中部分苹果会附带特殊状态效果，例如：

- 生命恢复 / 伤害吸收 / 抗性提升
- 火焰抗性 / 力量
- 发光
- 自定义效果：`天照`、`月读`

### 2. 肉类与食物加工

加入了更细分的肉类与食物：

- 小猪肉 / 熟小猪肉
- 小羊肉 / 熟小羊肉
- 小牛肉 / 熟小牛肉
- 小鸡肉 / 熟小鸡肉
- 肉沫
- 肉丸
- 汉堡

### 3. 其他内容

除了食物以外，还有一些拓展内容：

- 苹果块
- 苹果剑
- 自定义生物：`qzc`
- 自定义结构：苹果树
- 自定义状态效果贴图与实体资源

## 合成示例

仓库中已经包含数据生成后的配方文件，部分内容例如：

- `钻石苹果`：苹果 + 钻石
- `无限苹果`：由多种高级苹果合成
- 小型肉类可以进一步烧炼成熟肉

如果你想查看具体配方，可以直接看：

- `src/main/generated/data/more_food/recipe/`

## 项目结构

```text
src/main/java/com/nabr/morefood/
├─ effect/                 # 自定义效果
├─ entity/                 # 实体相关
├─ item/                   # 物品、方块、工具材料
├─ mixin/                  # Mixin 注入
└─ datagenertor/           # 数据生成

src/main/resources/
├─ assets/more_food/       # 贴图、语言文件、图标
└─ data/more_food/         # 结构等资源
```

## 开发与构建

### 本地构建

```bash
./gradlew build
```

构建完成后，产物通常会在：

```text
build/libs/
```

### 本地运行开发环境

```bash
./gradlew runClient
```

## 安装方法

1. 安装对应版本的 **Fabric Loader**
2. 安装 **Fabric API**
3. 将构建好的模组 `.jar` 放入 `.minecraft/mods`
4. 使用对应版本的 Minecraft 启动游戏
